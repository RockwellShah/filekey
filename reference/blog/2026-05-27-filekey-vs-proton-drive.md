Proton Drive is encrypted cloud storage from the makers of Proton Mail, with file sharing built in. FileKey is a focused tool for encrypting a file to one person. If you're choosing how to send something sensitive, here's how they compare.

## What Proton Drive does

Proton Drive stores your files encrypted in the cloud and lets you share them via a link, optionally with a password and an expiry. It's genuinely end-to-end encrypted and from a privacy-focused company. It's also a storage product: an account, apps, and your files living on Proton's servers (encrypted).

## How FileKey differs

FileKey isn't storage. It encrypts a file to your recipient's passkey on your device, and you send the file yourself, with no account and nothing stored anywhere. Proton Drive is the better home for files you want to keep and sync; FileKey is the leaner tool for the one-off "send this to this person, privately."

| | Proton Drive | FileKey |
| --- | --- | --- |
| What it is | Encrypted cloud storage and sharing | Encrypt a file to one person |
| Account | Proton account | None |
| Where files live | Proton's servers (encrypted) | Nowhere; you send the file |
| Sharing | Link, optional password and expiry | Encrypted to a passkey |
| Best for | Keeping and syncing files | Sending one file privately |

## Which to use

If you want encrypted storage and a place your files live, Proton Drive is excellent. If you just want to send one sensitive file to one person without an account or a server holding it, FileKey is the simpler fit.
